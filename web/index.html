Campus Pathfinder & Planner

Final Project Demo – Data Structures & Algorithms

Project Overview

This project simulates tools a student might use to navigate campus and plan their schedule. Under the hood, the C++ version of this project uses:

  • Lists – daily course schedules
  • Stacks & Queues – DFS, BFS and path reconstruction
  • Hash Tables – fast building & course lookup
  • Binary Search Tree (BST) – courses sorted by code
  • Graphs – campus buildings and connections
  • Merge Sort & Binary Search – course listing & search
  • Dijkstra’s Algorithm – shortest path between buildings

Use the tabs above to try the Campus Map and Schedule features in this browser demo.
